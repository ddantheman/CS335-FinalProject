To set up the Unity project with git, first install git-lfs at https://git-lfs.com/
Then, create a Unity Universal 3D project in your desired folder.
Within the project's base directory, run the commands:
* git lfs install
* git remote add origin https://github.com/ddantheman/CS335-FinalProject.git
* git branch -M main
If it says you have 10000+ changes, something probably went wrong. Make sure the repository directory matches your project directory when adding the remote branch.
